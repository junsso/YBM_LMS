문제유형 선택

객관식
  • 화면 클릭 : 보기 번호 배치
  • 보기 더블 클릭 : 정답 설정 (복수정답 가능)
  • 보기 드래그 : 위치 이동
  • 보기의 `X` 버튼 클릭 : 보기 삭제

선택형1
  • 화면 드래그 : 선택 영역 설정
  • 선택영역 클릭 : 정답 설정

선택형2
  • 화면 드래그 : 선택 영역 설정
  • 선택영역 클릭 : 정답 설정

매칭형
  • 화면 클릭&클릭 : 매칭할 시작점, 끝점 설정
  • 보기 드래그 : 위치 이동
  • 보기의 `X` 버튼 클릭 : 보기 삭제

단답형
  • 화면 드래그 : 입력창 설정
  • 입력창 클릭 : 정답 입력 (복수정답 “/”로 설정)
  • 입력창 드래그 : 이동
  • 입력창 X 버튼 클릭 : 삭제

서술형
  • 화면 드래그 : 입력창 설정
  • 입력창 클릭 : 정답 예시 입력
  • 입력창 드래그 : 이동
  • 입력창 X 버튼 클릭 : 삭제

도구입력
  • 화면 드래그 : 그리기 영역 설정
  • 영역 설정창 X 버튼 클릭 : 삭제

OX형
  • 화면 드래그 : 그리기 영역 설정
  • 영역 설정창 X 버튼 클릭 : 삭제

드롭다운형
  • 화면 드래그 : 선택 영역 설정
  • 팝업에서 드롭다운 상세 정보 설정

수식편집기
  • 화면 드래그 : 가리기 영역 설정
  • 영역 설정창 X 버튼 클릭 : 삭제

문제묶기
  • 화면 드래그 : 가리기 영역 설정
  • 영역 설정창 X 버튼 클릭 : 삭제

가리기
  • 화면 드래그 : 가리기 영역 설정
  • 영역 설정창 X 버튼 클릭 : 삭제

음원입력
  • 음원입력 클릭 : 음성 파일 첨부 또는 녹음가능 파일 저장 시 음성 아이콘 생성
  • 아이콘 드래그 : 위치 이동
  • 아이콘 X 버튼 클릭 : 삭제

10건의 첨부가 가능하며, 문항수엔 포함되지 않습니다.

링크연결

  • 아이콘 드래그 : 위치 이동
  • 아이콘 X 버튼 클릭 : 삭제
